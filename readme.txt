Для запуска backend необходимо найти в терминале папку с кодом и ввести команду python app.py. В файле requirements.txt в этой папке вы найдете все пакеты, использованные для backend. Если вы меняете порт в app.py, то вам нужно будет зайти в папку frontend в файл proxy.conf.json и изменить "target": "http://127.0.0.1:3000/api" на "target": "http://127.0.0.1:port/api", где port - заданный вами порт на backend.
