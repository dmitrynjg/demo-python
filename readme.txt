Для запуска backend необходимо войти в папку backend в терминале и ввести команду python app.py. В файле requirements.txt в этой папке вы найдете все пакеты, использованные для backend. Если вы меняете порт в app.py, то вам нужно будет зайти в папку frontend в файл proxy.conf.json и изменить "target": "http://127.0.0.1:3000/api" на "target": "http://127.0.0.1:port/api", где port - заданный вами порт на backend.

Для запуска frontend необходимо войти в папку frontend через терминал, установить все пакеты через команду npm i и написать команду npm run start. В терминале будет указан порт, на котором запущен сервер (обычно это 4200 порт, если он не занят). После этого вы сможете открыть frontend в браузере по адресу http://127.0.0.1:4200/.
